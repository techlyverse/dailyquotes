# Quotes App

**DailyQuotes is a clean, modern, and completely offline-first mobile application designed to deliver daily inspirational and motivational quotes. Built with Flutter, this app offers reliable, lightning-fast access to a curated collection of quotes without needing any internet connection.**

## Key Features

The app is built for seamless performance and reliable access, no matter the user's connectivity:

* **100% Offline Functionality:** All quotes and app features are available instantly, ensuring reliability even in airplane mode or areas with no signal.
* **Daily Quote Presentation:** Automatically presents a new, inspirational quote to help users start their day with a positive mindset.
* **Persistent Favorites:** Users can save their most resonant quotes to a personal collection, with data stored securely on the device.
* **Quote Sharing:** Easily share quotes as text or images with friends and family via device-level sharing features.
* **Intuitive UI/UX:** A beautiful and simple user interface built with Flutter for a smooth, native-like experience on all platforms.

## Getting Started (For Developers)

Follow these steps to set up the project on your local machine. Since the project is offline, the setup is much simpler!

### Prerequisites

You only need the following installed:

1.  **Flutter SDK:** [Installation Guide](https://flutter.dev/docs/get-started/install)
2.  **Dart SDK:** (Included with Flutter)
3.  **Code Editor:** VS Code or Android Studio with Flutter/Dart plugins.

### Local Installation

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/techlyverse/dailyquotes.git](https://github.com/techlyverse/dailyquotes.git)
    cd dailyquotes
    ```

2.  **Install Dependencies**
    ```bash
    flutter pub get
    ```

3.  **Run the App**
    Connect a device or start an emulator/simulator, and run the following command:
    ```bash
    flutter run
    ```
    The application will run immediately without any external backend configuration.
